# 安全 Skill

网站项目必须重点关注安全。

## 密码处理

- 密码必须使用哈希存储
- 绝对不能明文存储密码

## 鉴权

建议：

- JWT + HttpOnly Cookie
或
- Session 方案

优先选择简单、稳定、易维护的方案。

## 服务端校验

以下逻辑必须在后端做：

- 登录态校验
- 权限判断
- 参数校验
- 用户身份判断

## 环境变量

敏感信息必须放到服务端环境变量中，例如：

- 数据库连接串
- JWT 密钥
- 第三方服务密钥

## 前端限制

- 前端校验只能提升体验，不能替代后端校验
- 页面层不要信任浏览器传来的任何敏感状态
