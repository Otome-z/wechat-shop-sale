# 任务开始模板

请按以下流程工作：

1. 先阅读 `AGENTS.md`
2. 再阅读相关 `.codex` 文档
3. 先做需求拆解
4. 再做技术设计
5. 用户确认后再开发
6. 最后做 review

当前项目技术栈约定为：

前端：

- Vue 3
- uni-app
- TypeScript
- Pinia
- Axios
- Sass / SCSS
- ESLint
- Prettier

后端：

- Node.js
- Express
- TypeScript
- Prisma

数据库：

- MySQL

工程化：

- Husky
- lint-staged

默认不包含：

- 发布
- 部署
- CI/CD
- 管理后台
